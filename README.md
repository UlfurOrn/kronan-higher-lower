# Hærra eða Lægra? 🛒

Higher or Lower game using real grocery prices from [Krónan](https://kronan.is) — Iceland's supermarket chain.

Players compare two products and guess which one has the higher **price per unit** (kr/kg, kr/l, or kr/stk).

---

## Features

- **Normal mode** — 3 lives
- **Hard mode** — 1 life
- **Category filter** — play with all products or just one category
- **Streak tracking** — best streak persisted in browser localStorage
- **Icelandic UI** — fully in Icelandic with real Krónan product names and prices

---

## Prerequisites

- [Node.js LTS](https://nodejs.org/) (v20 or later)
- A Krónan account with an API access token (for refreshing product data)

## Getting a Krónan API Token

1. Log in at [kronan.is](https://kronan.is) with Auðkenni
2. Go to **Stillingar** (Settings) → **API tókar**
3. Create a new token and copy it

The token is only needed to run the data pipeline — not to play the game.

---

## Refreshing Product Data

The game uses a pre-generated `src/data/products.json` file. To fetch fresh data from the Krónan API:

```bash
KRONAN_API_TOKEN=your_token_here npm run fetch-data
```

This script will:
- Fetch all products from the Krónan catalog
- Filter to only products with an image, a price, and a computable price-per-unit
- Write `src/data/products.json`
- Print a summary of how many products were included/excluded

> **Recommendation**: Re-run this weekly or before any deployment to keep prices current.

A sample `src/data/products.json` with 12 placeholder products is included so you can run the app without a token during development.

## How It Works

1. **Data pipeline** (`npm run fetch-data`) queries the Krónan Public API and produces `products.json`
2. **React app** loads `products.json` at startup — no runtime API calls
3. Each round: two products are shown, one with its price revealed, one hidden
4. The player guesses **Hærra** (Higher) or **Lægra** (Lower)
5. Wrong answers cost a life; the game ends when all lives are gone
6. The best streak is saved in `localStorage` and shown on the Game Over screen
